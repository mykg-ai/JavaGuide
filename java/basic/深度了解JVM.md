## 编译过程
创建文件`HelloWorld.java`
```
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```
终端运行命令，生成`HelloWorld.class`文件
```
$ javac HelloWorld.java
```
终端运行命令，执行文件
```
$ java HelloWorld
```
